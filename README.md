# SQLite Simplifier

A powerful, type-safe SQL query builder with built-in relation support, query caching, and advanced features for SQLite.

## Features

- 🔍 **Type-safe queries** with full TypeScript support
- 🚀 **Fluent query builder** with intuitive API
- 🔗 **Relationship support** (LEFT, INNER, RIGHT joins)
- 📊 **Aggregate functions** (COUNT, SUM, AVG, MIN, MAX)
- 🎯 **Advanced filtering** with complex WHERE conditions
- 💾 **Query caching** to prevent duplicate requests
- 📝 **Comprehensive logging** for debugging
- 🛡️ **SQL injection protection** with parameterized queries

## Installation

```bash
npm install sqlite-simplifier
# or
yarn add sqlite-simplifier
# or
pnpm add sqlite-simplifier
```
